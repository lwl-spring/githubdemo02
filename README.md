## 中间遇到的问题
### swagger2 2.2.2版本与spring cloud feign 冲突，导致feign接口注入bean为null的问题
  解决办法：
    1. 如果swagger版本是v1.x，那么请参考：https://segmentfault.com/a/1190000006595187。
    2. 如果swagger版本是v2.x，那么请将升级swagger版本到2.5.0以上即可。
